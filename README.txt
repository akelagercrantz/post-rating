A rating plugin that aims allows developers to access information such as top rated posts.

== Description ==

Post rating is a review rating plugin that can display rating by stars or in text. You can define the maximum rating yourself,
or use the default scale of 5. Being a developer you can easily access the top rated posts, so you can write widgets for your
theme.

Recommended usage with the plugin hreview.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the post_rating() template function in your template files where you want the rating to de displayed.
